## 🚀 快速开始

### 前置要求

**Python 版本**: Python 3.10（推荐）

**安装 uv** (如果尚未安装):

```bash
# 方法 1: 使用 pip 安装（推荐）
pip install uv

# 方法 2: 使用官方脚本
curl -LsSf https://astral.sh/uv/install.sh | sh

# 或者运行提供的安装脚本
./install_uv.sh
```

### 方式一：使用 uv + Makefile（推荐）

```bash
# 一键完成所有步骤（创建虚拟环境 + 安装依赖 + 生成数据 + 运行检测）
make all

# 或者分步执行
make venv     # 使用 uv 创建虚拟环境
make deps     # 安装依赖包到虚拟环境
make data     # 生成模拟数据
make run      # 运行异常检测主程序

# 其他有用命令
make test     # 测试环境配置
make clean    # 清理生成的文件（可选：rm -rf .venv 删除虚拟环境）
make help     # 显示帮助信息
```

**激活虚拟环境**:
```bash
# macOS/Linux
source .venv/bin/activate

# Windows
.venv\Scripts\activate
```

## 🔧 故障排查

### 问题 1：中文字体显示异常
```python
# 如果遇到中文显示问题，尝试修改字体配置
plt.rcParams['font.sans-serif'] = ['Arial Unicode MS']  # macOS
plt.rcParams['font.sans-serif'] = ['SimHei']  # Windows
plt.rcParams['font.sans-serif'] = ['DejaVu Sans']  # Linux
```

### 问题 2：未检测到异常
- 检查数据是否包含明显的异常点
- 尝试降低 Sigma 阈值（如改为 2.5）
- 确认数据文件正确加载

### 问题 3：图片无法保存
- 检查当前目录是否有写权限
- 确保 matplotlib 后端配置正确
- 可以尝试注释掉 `plt.show()` 只保存图片

## ❓ 常见问题

**Q: 为什么选择 3-Sigma 而不是其他方法？**  
A: 3-Sigma 简单快速，无需训练，适用于稳定的指标。对于复杂场景可以结合机器学习方法。

**Q: 如何确定合适的 Sigma 值？**  
A: 通常使用 3，可以根据业务容忍度调整。敏感场景用 2.5，宽松场景用 3.5。

**Q: 可以检测负向异常吗？**  
A: 可以！代码已经支持检测低于下界的异常情况。
