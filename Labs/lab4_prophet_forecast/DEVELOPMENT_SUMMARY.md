## 🔧 遇到的问题与解决方案

### 问题 1: pyproject.toml 版本格式错误
**现象**: 
```
TOML parse error: requires-python = "=3.10"
Failed to parse version: no such comparison operator "="
```

**解决**: 
```toml
# 错误
requires-python = "=3.10"

# 正确
requires-python = ">=3.10"
```

---

### 问题 2: Prophet API 变更
**现象**: 
```
TypeError: Prophet.plot_components() got an unexpected keyword argument 'ax'
```

**原因**: Prophet 1.3 版本移除了 `ax` 参数支持

**解决**: 添加 try-except 块，提供降级方案
```python
try:
    model.plot_components(forecast)
except Exception as e:
    # 手动绘制组件
    ax1.plot(forecast['ds'], forecast['trend'], label='趋势')
```

---

### 问题 3: numpy 索引类型错误
**现象**: 
```
IndexError: only integers, slices (`:`), ellipsis (`...`) are valid indices
```

**原因**: 使用浮点数进行数组索引

**解决**: 使用整数索引
```python
# 错误
actual_dates[i-0.5]

# 正确
actual_dates[max(0, i-1)]
```
