## 🎯 详细实验目标

### Lab 1: 3-Sigma 异常检测

**难度**: ⭐⭐☆☆☆ | **时间**: 45-60 分钟

学完本实验，你将能够：

1. ✅ **理解 3-Sigma 原则的统计学原理和应用场景**
   - 掌握正态分布的特性
   - 理解均值和标准差的统计意义
   - 识别适用于 3-Sigma 的场景

2. ✅ **掌握基于均值和标准差的异常检测方法**
   - 计算滚动均值和标准差
   - 设定动态阈值（±3σ）
   - 判定异常点

3. ✅ **学会计算动态上下界并识别异常点**
   - 实现滑动窗口机制
   - 实时计算置信区间
   - 标记超出边界的异常

4. ✅ **掌握异常检测结果的可视化与报告生成**
   - 绘制原始数据 + 上下界曲线
   - 标注异常点和置信区间
   - 生成专业检测报告

5. ✅ **能够使用滑动窗口处理非平稳序列**
   - 理解平稳与非平稳序列的区别
   - 应用滑动窗口使序列平稳化
   - 选择合适的窗口大小

---

### Lab 2: Isolation Forest 多维异常检测

**难度**: ⭐⭐⭐☆☆ | **时间**: 60-90 分钟

学完本实验，你将能够：

1. ✅ **理解 Isolation Forest 的"孤立"思想和算法原理**
   - 掌握随机分割的孤立化过程
   - 理解路径长度与异常得分的关系
   - 对比孤立森林 vs 传统方法

2. ✅ **掌握使用 scikit-learn 实现 Isolation Forest 模型**
   - 正确配置模型参数
   - 训练和预测流程
   - 解释 anomaly_score 和 labels

3. ✅ **学会对 CPU、内存、磁盘等多维指标进行联合异常检测**
   - 特征工程和多维数据准备
   - 联合检测的优势
   - 识别单变量无法发现的异常

4. ✅ **掌握 contamination 参数调优方法**
   - 理解污染率的含义
   - 基于业务先验知识设置参数
   - 通过网格搜索优化参数

5. ✅ **能够使用 PCA 降维可视化高维异常分布**
   - 将高维数据降至 2D/3D
   - 可视化正常点和异常点的分布
   - 解释降维后的聚类效果

6. ✅ **理解多维检测相比单变量检测的优势**
   - 识别特征间的关联异常
   - 发现隐蔽的复合故障
   - 降低误报率和漏报率

---

### Lab 3: LSTM Autoencoder 时序异常检测

**难度**: ⭐⭐⭐⭐☆ | **时间**: 90-120 分钟

学完本实验，你将能够：

1. ✅ **理解 LSTM 网络的门控机制和长期依赖捕捉能力**
   - 掌握遗忘门、输入门、输出门的作用
   - 理解细胞状态的信息传递
   - 对比 LSTM vs RNN 的优势

2. ✅ **掌握自动编码器的编码器 - 解码器架构设计**
   - 理解压缩和重构的思想
   - 设计瓶颈层（bottleneck）
   - 训练无监督重构模型

3. ✅ **学会基于重构误差的无监督异常检测方法**
   - 计算 MSE/MAE 重构误差
   - 设定自适应阈值
   - 无需标注数据即可检测异常

4. ✅ **掌握时序数据的滑动窗口预处理技术**
   - 构建 sequence-to-sequence 样本
   - 选择合适的 sequence_length
   - 处理边界效应

5. ✅ **能够调优深度学习模型的超参数**
   - hidden_size、num_layers 选择
   - batch_size、learning_rate 调优
   - 防止过拟合的策略（Dropout、正则化）

6. ✅ **理解深度学习模型在复杂时序模式识别中的优势**
   - 捕捉非线性关系
   - 适应多变的时序模式
   - 自动特征提取

---

### Lab 4: Prophet 时序预测与异常检测

**难度**: ⭐⭐⭐☆☆ | **时间**: 60-90 分钟

学完本实验，你将能够：

1. ✅ **掌握 Prophet 时序分解方法（趋势 + 季节 + 节假日）**
   - 理解加法模型和乘法模型
   - 分解趋势项、季节项、节假日项
   - 可视化各成分贡献

2. ✅ **学会使用置信区间进行异常判定**
   - 理解不确定性采样
   - 设置 interval_width
   - 识别超出置信区间的异常点

3. ✅ **理解预测残差分析与 Z-Score 应用**
   - 计算实际值与预测值的残差
   - 对残差进行 Z-Score 标准化
   - 基于统计阈值判定异常

4. ✅ **能够对比 Prophet vs LSTM 的适用场景**
   - Prophet: 强季节性、可解释性强
   - LSTM: 复杂模式、非线性关系
   - 根据数据特点选择合适模型

---

### Lab 5: One-Class SVM 未知故障模式识别

**难度**: ⭐⭐⭐⭐☆ | **时间**: 75-90 分钟

学完本实验，你将能够：

1. ✅ **理解 One-Class SVM 的边界检测思想**
   - 掌握最大间隔分类器原理
   - 理解支持向量的作用
   - 区分 One-Class vs 传统 SVM

2. ✅ **掌握核函数（RBF、Linear）的选择策略**
   - 理解核技巧和高维映射
   - RBF 核的参数调优（gamma）
   - 根据数据分布选择核函数

3. ✅ **学会处理小样本/不平衡数据场景**
   - 只有正常样本的训练策略
   - nu 参数的含义和设置
   - 应对类别不平衡

4. ✅ **能够应用于未知故障类型的识别**
   - 检测训练集中未出现的异常
   - 开放集识别（Open Set Recognition）
   - 适用于新型攻击、未知故障

### Lab 6: 基于 RAG 的故障知识问答系统

**难度**: ⭐⭐⭐⭐⭐ | **时间**: 120-150 分钟

学完本实验，你将能够：

1. ✅ **理解 RAG 架构和检索增强生成原理**
   - 掌握检索 + 生成的双阶段流程
   - 理解为什么需要外部知识
   - 对比 RAG vs Fine-tuning

2. ✅ **掌握文本嵌入（Embedding）技术**

   - 多语言嵌入的处理

3. ✅ **学会使用 FAISS 进行向量相似度搜索**
   - 构建向量索引（IndexFlatIP、IndexIVF）
   - 归一化与余弦相似度
   - Top-K 检索和阈值过滤

4. ✅ **能够构建基于向量检索的智能问答系统**
   - 知识库的构建和更新
   - 上下文感知答案生成
   - 处理多轮对话和追问

## 🎓 学习建议

### 循序渐进
```
Lab 1 (基础) → Lab 2 (进阶) → Lab 3 (提高) → ...
```
不要跳级学习，每个实验都是后续的基础。

### 动手实践
```bash
# 每个实验都要亲自运行
cd labX_xxx
make all

# 修改参数，观察变化
# 添加新功能，扩展实验
```

### 深度思考
- 这个方法的优缺点是什么？
- 在实际生产中会遇到什么问题？
- 如何改进和优化？
